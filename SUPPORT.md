# Getting Help with maxentcpp

## Where to ask questions

- **GitHub Discussions**: <https://github.com/alrobles/maxentcpp/discussions>
- **GitHub Issues**: <https://github.com/alrobles/maxentcpp/issues> (for bugs
  and feature requests)

## Reporting bugs

When filing a bug report, please include:

1. A minimal reproducible example (use `reprex::reprex()` if possible)
2. Output of `sessionInfo()`
3. Operating system and R version
4. The exact error message or unexpected output
5. What you expected to happen

## Supported platforms

`maxentcpp` is tested on:

- Ubuntu (latest), macOS (latest), Windows (latest)
- R >= 3.5.0
- C++17 compilers: GCC >= 7, Clang >= 5, MSVC >= 2017
